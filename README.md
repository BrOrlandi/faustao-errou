# Faustão Errou!!

<p align="center">
    <img src="./errou.gif"/>
</p>

> Errouuuuuu!

Play famous "errrrouuu" sound by Faustão.

## Compatibility

- [x] Linux
- [x] Windows
- [x] Mac

## Installing
Install globally:

    npm install -g faustao-errou

Or install locally:

    npm install faustao-errou

## Running
Run command:

    errou

Or import in NodeJS script:

    var faustaoErrou = require('faustao-errou');

    faustaoErrou();
